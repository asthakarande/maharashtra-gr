📄 Maharashtra GR Portal (Unofficial)

A web-based platform to search and access **Government Resolutions (GRs)** issued by Maharashtra state departments.

> ⚠️ This is an independent project and is **not affiliated with the Government of Maharashtra. All documents are sourced from official government websites.

---

🚀 Features

* 🔍 Search GRs by keyword
* 🏢 Filter by department
* 📅 Filter by date range
* 📄 View GR PDFs directly
* ⬇️ Download GR documents
* 📑 Pagination for large datasets

---

📊 Dataset

* 5000+ Government Resolutions
* Multiple departments covered (e.g., Public Works Department)
* PDF links sourced from official government portals

---

🛠️ Tech Stack

* HTML – Structure
* CSS – Styling & UI
* JavaScript – Filtering, rendering, pagination

---

⚙️ How It Works

1. User selects filters (department, keyword, date)
2. Data is filtered using JavaScript
3. Matching GRs are displayed with pagination
4. Users can view or download PDFs directly
